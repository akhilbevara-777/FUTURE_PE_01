# FUTURE_PE_01

## Project Title
AI Website Copy Generator for Local Businesses

## Business Chosen
GlowUp Salon – Hyderabad

## Objective
To create AI-generated website copy for a local salon business using Prompt Engineering techniques.

## Tools Used
- ChatGPT
- GitHub

## Files Included
- Homepage Copy
- Services Page Content
- CTA Sections
- Structured Prompts

## Prompt Logic
Prompts were designed to generate:
- Clear value propositions
- Business-specific language
- Persuasive call-to-actions
- Professional and user-friendly website content

## Outcome
Created a complete website copy set ready for real-world salon websites.
